# 🐍 双人在线贪吃蛇游戏

基于原生 JavaScript + Socket.io + Vite 构建的双人在线对战贪吃蛇游戏。

## ✨ 功能特性

- 🏠 **创建/加入房间** - 通过6位房间码邀请好友对战
- 🎮 **实时对战** - 流畅的多人游戏体验
- 💬 **内置聊天** - 游戏中与好友聊天
- 🏆 **排行榜** - 记录高分玩家
- 🎨 **精美界面** - 现代化的 UI 设计

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动服务器和客户端

```bash
# 方式一：分别启动
# 终端1：启动后端服务器
npm run server

# 终端2：启动前端开发服务器
npm run dev

# 方式二：同时启动（需要安装 concurrently）
npm start
```

### 访问游戏

- 前端地址：`http://localhost:5173`
- 后端地址：`http://localhost:3000`

## 🎯 游戏玩法

1. **创建房间** - 输入你的名字，点击"创建房间"
2. **分享房间码** - 将6位房间码分享给好友
3. **加入房间** - 好友输入名字和房间码加入
4. **开始游戏** - 两人都准备好后，房主点击"开始游戏"
5. **控制蛇** - 使用方向键(↑↓←→)或WASD控制方向
6. **吃食物得分** - 吃到红色食物获得10分
7. **获胜条件** - 对手撞墙或撞到自己/你的蛇，你就获胜

## 🔧 技术栈

- **前端**：原生 JavaScript + Canvas API + Vite
- **后端**：Node.js + Express + Socket.io
- **通信**：WebSocket 实时通信

## 🎮 操作说明

| 操作 | 按键 |
|------|------|
| 向上 | ↑ 或 W |
| 向下 | ↓ 或 S |
| 向左 | ← 或 A |
| 向右 | → 或 D |
| 发送消息 | Enter |

## 📝 游戏规则

1. 每吃到一个食物得10分
2. 蛇撞到墙壁、自己或对方蛇即失败
3. 如果双方同时死亡，分数高者获胜
4. 分数相同则为平局
